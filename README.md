# Speech Emotion Recognition 🎤

## Overview
This project detects human emotions from speech audio.

## Dataset
RAVDESS dataset

## Model
CNN + LSTM (TimeDistributed Conv1D)

## Features
MFCC (Mel-Frequency Cepstral Coefficients)

## Emotions Classified
- Happy
- Sad
- Angry
- Fearful
- Neutral
- Calm
- Disgust
- Surprised

## Technologies that is used 
- Python
- TensorFlow / Keras
- Librosa

## Result
Model achieved good accuracy in classifying emotions which is 75%.

## How to Run
1. Install requirements
2. Run the notebook
